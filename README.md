<h1 align="center">VEIL Windows GUI</h1>

<p align="center">A quiet Windows client for the VEIL private tunnel: Tauri desktop app + Go sidecar.</p>

---

`veil-windows-gui` is the user-facing Windows client: one connection state,
the active endpoint, traffic counters, last handshake, split-tunnel editing,
logs, and config selection.

## Components

- `src/` - Vue 3 + TypeScript + shadcn-vue frontend (dark theme, VEIL brand
  palette), built with Vite.
- `src-tauri/` - Tauri (Rust) desktop shell. Spawns `veil-sidecar.exe` as a
  child process and drives it over stdin/stdout using the control protocol
  from [veil-windows](https://github.com/veil-proto/veil-windows)'s `control`
  package (newline-delimited JSON). Requests Administrator on launch (TUN
  adapter creation and routing changes need it).
- `cmd/veil-sidecar` - Go binary that owns the TUN adapter, routing, DNS, UDP
  socket, and VEIL engine (via `veil-windows`'s `wintunnel` package), and
  serves the control protocol on its own stdio. No Windows Service, no named
  pipe: the tunnel lives only as long as the app is open.

## Install

Download the latest installer from
[Releases](https://github.com/veil-proto/veil-windows-gui/releases) and run
it, then open **VEIL** from the Start menu. Windows will prompt for
Administrator on launch — that's expected, and needed for the tunnel adapter.

## Import A Config

The GUI accepts full `veil://` config links generated by
[veil-install](https://github.com/veil-proto/veil-install):

1. Copy a `veil://...#Name` link.
2. Open VEIL.
3. Paste it into the import field and press **Import link**.
4. Select the imported config and press **Connect**.

Imported configs are stored under `%APPDATA%\VEIL\configs` as plain `.conf`
files. The sidecar persists the active config under `%ProgramData%\VEIL` so
reopening the app reconnects the last tunnel.

## Development

```powershell
# Frontend
npm install
npm run dev            # Vite dev server

# Sidecar (cross-compiled from Linux with mingw, or natively on Windows)
go build -o veil-sidecar.exe ./cmd/veil-sidecar

# Full app (spawns the sidecar as a Tauri "externalBin")
npx tauri dev
npx tauri build
```
